# Hellwave Game Code

This is a mod I wrote to validate how easy it is to write a mod on top of the reimplementation of the Quake 1 game code in JavaScript.

## Concept

This mod is a PVE co-op game similiar to titles like Killing Floor. You have to defend yourself against a horde of enemies and each killed enemy will yield in-game money you can use to buy weapon upgrades between waves of enemies.
